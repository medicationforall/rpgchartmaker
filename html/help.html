HelpX
    List
  • "Add List" creates a basic list.
  • Pressing the enter key on the List Entry label adds it to the list.
  • Click on a list item to edit it.
  • Drag list items to re-order them.
  • You can drag items across lists.
  • Click the "A-Z" to alphabetize a list.
    Object List
  • "Add object List" creates a complicated list.
  • Before you can add list items you have to make form inputs.
  • Enter key press on Label will add it to the form.
  • Edit a form input by clicking on it's label.
  • Enter key will apply changes to a form inputs label.
  • Drag form inputs to re-order them.
